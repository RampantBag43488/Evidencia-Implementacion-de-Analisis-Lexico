## Lenguaje seleccionado

El lenguaje seleccionado para esta evidencia es un subconjunto de **HTML text-related tags**.

HTML es un lenguaje de marcado utilizado para estructurar contenido en páginas web. Para esta evidencia no se analiza todo HTML, sino únicamente un conjunto limitado de etiquetas relacionadas con texto.

El lenguaje definido acepta etiquetas de apertura y cierre con la siguiente forma:
- `<etiqueta>`
- `</etiqueta>`

Donde etiqueta pertenece al conjunto: p, h1, h2, h3, h4, h5, h6, strong, em, span, mark, small, sub, sup.
